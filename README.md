Fruit Detection and Classification System

This project is a machine learning-based system developed in MATLAB for detecting and classifying fruits in real time.

Features
•  Image segmentation and feature extraction
•  Fruit classification using machine learning techniques
•  Graphical User Interface (GUI) for displaying classification results
•  Real-time image capture using IP camera

Technologies
•  MATLAB
•  Image Processing Toolbox
•  Machine Learning techniques

Dataset
Images of fruits such as watermelon, apple, and banana were used for training and testing the classification system.
